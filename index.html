ARMY GAME




RULES


  • By default there will be two teams Terrorist and Counter Terrorist.


  • Terrorist need to score 16 points in order to conquer the State.


  • There are total of 64 cities in the entire state.


  • Terrorist can plant there base camp in the neighbour cities only in order to spreads terror.


  • If Terrorist are not able to gain 16 neighbour cities then ultimately Counter Terrorist wins.
